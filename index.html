Este texto utiliza, como referencia, fundamentalmente, la biblioteca estándar de Java, con abundantes enlaces a sus clases y métodos: está pensado con el objetivo de familiarizarse con la misma y su lectura. Por razones del momento en que estos apuntes germinaron, dichos enlaces son a la versión 14. Utiliza los tutoriales de Java, en particular, los que aluden a excepciones (para el capítulo 3) y concurrencia.

También, múltiples libros y contenidos que han constituido la base de conocimiento del autor, aunque tal vez no se mencionen porque ya no se recuerda la fuente en sí, así como diferentes contenidos accesibles en Internet: algunos están especificados pero desafortunadamente, de otros, se ha perdido la cita.

Los capítulos dedicados a la programación paralela y concurrente tienen, como precedente, los cursos Parallel Programming in Java y Concurrent Programming in Java del profesor Vivek Sarkar disponibles en Coursera:

  • Parallel Programming in Java.
  • Concurrent Programming in Java.
